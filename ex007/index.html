LISTA ORDENADA

a lista ordenada é usada com a tag ol, que pode criar listas
com items ordenados, e é obrigatorio usar a tag li pois
significa list-item!

a tag também tem um parametro chamado type e ele tem 4 valores
disponiveis

eles são
  • 1-ordem númerica
  • a-ordem alfabetica minuscula
  • A-ordem alfabetica maiuscula
  • I- ordem romana maiuscula
  • i- ordem romana minuscula!

LISTA NÃO ORDENADA

A lista não ordenada não possue uma ordem e é caracterizada pela tag
ul!
